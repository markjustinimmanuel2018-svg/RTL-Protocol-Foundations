**# RTL Mini Projects - Communication Protocols**



This repository contains a collection of communication protocol mini projects focusing on understanding serial communication protocols, protocol timing, FSM-based control logic, and verification methodologies.



\-> This repository represents **Phase-2 Cycle-1: RTL Protocol Foundations** of my RTL learning journey, focused on implementing and verifying industry-standard communication interfaces using synthesizable RTL design.



The goal of this repository is to:

* Understand serial communication protocols
* Implement protocol transmitters and receivers using RTL 
* Learn FSM-based protocol control
* Develop self-checking testbenches for verification
* Build a protocol-focused RTL portfolio for front-end VLSI roles
